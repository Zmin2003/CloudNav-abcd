# CloudNav (云航) - 轻量私有导航站

**本项目基于 [aabacada](https://github.com/aabacada) 的 [CloudNav-abcd](https://github.com/aabacada/CloudNav-abcd) 开源项目进行简化修改。**

---

## ✨ 主要特性

- 🚀 **极简部署** - 基于 Cloudflare Pages + KV，无需服务器，完全免费
- 📱 **响应式设计** - 完美适配桌面端和移动端
- 🔒 **密码保护** - 支持全局访问密码和分类密码
- ☁️ **多端同步** - 利用 Cloudflare KV 边缘存储，数据实时同步
- 📦 **WebDAV 备份** - 支持坚果云等 WebDAV 网盘备份
- 🔍 **站外搜索** - 可配置多个搜索引擎快速搜索
- 🎨 **深色模式** - 支持浅色/深色主题切换

---

## 🚀 快速部署

1. **Fork 项目** - 点击右上角 Fork 按钮
2. **创建 Pages 应用** - Cloudflare Dashboard → Workers & Pages → 创建应用程序 → Pages → 连接 Git
3. **配置构建**:
   - 构建命令: `npm run build`
   - 输出目录: `dist`
4. **创建 KV 数据库** - Workers & Pages → KV → 创建命名空间 `CLOUDNAV_DB`
5. **绑定变量**:
   - 绑定 KV: 变量名 `CLOUDNAV_KV`，选择 `CLOUDNAV_DB`
   - 环境变量: `PASSWORD` = 您的访问密码
6. **重新部署** - 完成！

## ⚙️ 使用说明

### Chrome 扩展程序
1. 点击设置 → 扩展工具
2. 复制生成的 3 个文件代码
3. 在 Chrome 扩展管理页开启开发者模式
4. 加载已解压的扩展程序

### 书签导入
支持导入 Chrome/Edge/Firefox 书签 HTML 文件，自动去重。

### WebDAV 备份
在备份页面配置 WebDAV 信息（支持坚果云、InfiniCloud 等），一键同步。
